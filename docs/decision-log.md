# 意思決定ログ（ADR: Architecture Decision Records）

「なぜこの設計・技術を選んだか」を記録します。
後続プロジェクトや将来の自分が「なぜこうなっているか」を理解できるようにします。

---

## テンプレート

```
## ADR-{番号}: {タイトル}

**日付:** YYYY-MM-DD  
**ステータス:** 提案中 / 承認済み / 廃止

### 背景
なぜこの決定が必要だったか

### 検討した選択肢
1. 選択肢A - メリット/デメリット
2. 選択肢B - メリット/デメリット

### 決定
選択肢Xを採用する

### 理由
採用した理由

### 影響・トレードオフ
この決定による影響、諦めたこと
```
