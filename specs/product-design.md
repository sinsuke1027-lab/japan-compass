## 1. プロダクトビジョン

> 日本を訪れる旅行者を「観光客」から「敬意ある訪問者」に変える。
> 文化を学び、作法を実践し、地域に貢献する——そのすべてをポケットの中に。

### 差別化軸

Google Maps や一般的な観光アプリとの違いは**サステナブル観光への特化**。
「どこに行くか」ではなく「どう在るか」を支援するアプリ。

| 一般観光アプリ | Japan Compass |
|---|---|
| スポット検索・経路案内 | 文化的背景まで含む作法・マナーの解説 |
| レビュー・評価 | サステナブルな行動（参拝・寄付・エコ活動）の記録 |
| 訪問数・チェックイン | 地域への貢献体験のジャーナル |

## 2. ターゲットユーザー

### メインペルソナ

**A. 初回訪日の文化リスペクター**
- 30〜40代、欧米・東南アジア出身
- 「日本で失礼なことをしたくない」という不安を持つ
- 旅の前に下調べをしっかりする
- 神社・寺院・温泉など文化体験に興味がある

**B. リピーター深堀りタイプ**
- 日本が好きで2回目以降の訪問
- 「今度はもっとローカルな体験をしたい」
- お賽銭・御朱印など、前回できなかったことに挑戦したい
- サステナブルな旅に意識がある

### 共通の価値観
- 旅先の文化を尊重したい
- 地域に何らかの形で貢献したい
- SNS 映えより「本物の体験」を求める

## 3. ユーザーストーリー

### 文化・マナー

| ID | ストーリー | 受け入れ条件 |
|---|---|---|
| US-001 | 初めて電車に乗る旅行者として、乗車マナーをシーン別に確認したい。恥ずかしい思いをしたくないから。 | 電車・優先席・荷物など項目別に英語で表示される |
| US-002 | レストランで注文前に、その席でのマナー（箸の使い方・割り勘など）を確認したい。 | 和食・ラーメン・居酒屋などシーン別に分岐できる |
| US-003 | 温泉施設の前で、入浴作法と禁止事項をその場ですぐ確認したい。 | 30秒以内に必要な情報にたどり着ける |

### 神社・寺院参拝

| ID | ストーリー | 受け入れ条件 |
|---|---|---|
| US-010 | 神社の鳥居の前で、参拝の正しい手順をステップバイステップで教えてほしい。 | 手水・お辞儀・鈴・お賽銭・二拝二拍手一拝の手順が画像付きで表示 |
| US-011 | お賽銭の意味と相場を英語で理解したい。意味を知らずにお金を投げたくないから。 | 金額の考え方・意味・注意点が説明される |
| US-012 | 参拝後、「今日ここで参拝してお賽銭を納めた」という体験を記録したい。 | 場所・日付・金額（任意）をメモできる |

### サステナブル活動

| ID | ストーリー | 受け入れ条件 |
|---|---|---|
| US-020 | 近くで地域貢献できる場所（寄付・エコ活動・清掃など）を地図で探したい。 | 現在地から半径 X km のスポットが表示される |
| US-021 | 旅行中に行ったサステナブルな活動を一覧で振り返りたい。 | 日付・場所・活動種別で一覧表示される |
| US-022 | 帰国後に「自分はどれだけ地域に貢献できたか」を実感したい。 | 旅全体のサマリー（訪問数・寄付合計など）が見える |

### 緊急・安全

| ID | ストーリー | 受け入れ条件 |
|---|---|---|
| US-030 | 体調が悪くなったとき、近くの病院と緊急連絡先を即座に確認したい。 | 3タップ以内で緊急情報にアクセスできる |
| US-031 | 日本語が読めない状態で警察を呼ぶ方法を確認したい。 | 110/119 の説明と英語で話す方法が表示される |

                                            → 確認して安心              「また来たい」

### 重点フェーズ

MVP では **来日前〜滞在中** の体験品質を最優先にする。
来日前: コンテンツの充実・オフライン対応
滞在中: 速さ（3タップ以内）・オフライン動作・記録機能

## 5. コアループ（アプリ内行動サイクル）

```
     ┌─────────────────────────────────────┐
     │                                     │
  [学ぶ]                               [記録する]
  マナー・作法を                        体験・寄付を
  読んで理解する                        ジャーナルに残す
     │                                     │
     └──────────→ [実践する] ──────────────┘
                  神社参拝・寄付・
                  エコ活動を現地で行う
```

このループが回るほど「日本への理解が深まった」実感が得られ、
リピート利用・次回来日への動機につながる。
