# デプロイ前 チェックリスト

デプロイ実行前に全項目を確認してください。

---

## コード品質
- [ ] 全テストがパスしている
- [ ] リント・フォーマットエラーがない
- [ ] デバッグ用の`console.log`・`print`が残っていない

## セキュリティ
- [ ] `.env`ファイルがコミットされていない
- [ ] シークレットがコードに直接書かれていない
- [ ] GCP Secret Managerに必要な値が登録されている
- [ ] セキュリティチェックリスト（`security-check.md`）完了済み

## 設定・環境
- [ ] 環境変数が本番用に設定されている
- [ ] デプロイ先GCPプロジェクトが正しい（誤って開発環境にデプロイしない）
- [ ] データベースのマイグレーションが適用済み（または計画済み）

## バックアップ・ロールバック
- [ ] 本番DBのバックアップを確認した
- [ ] ロールバック手順を把握している
- [ ] デプロイ後に問題があった場合の連絡フローを確認した

## デプロイ後確認
- [ ] スモークテストの実行確認（`post-deploy-smoke-test.yml`）
- [ ] ヘルスチェックエンドポイントが正常応答
- [ ] ログにエラーが出ていない（Cloud Logging確認）
- [ ] Slack通知が届いている（`notify.yml`）
