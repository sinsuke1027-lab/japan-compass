# セキュリティ チェックリスト

---

## 認証・認可
- [ ] 全APIエンドポイントに適切な認証が実装されている
- [ ] 権限のないリソースへのアクセスが拒否される
- [ ] セッション・トークンの有効期限が設定されている

## 入力検証
- [ ] 全ユーザー入力にバリデーションが実装されている
- [ ] SQLインジェクション対策（プリペアドステートメント使用）
- [ ] XSS対策（出力エスケープ）
- [ ] CSRF対策

## データ保護
- [ ] パスワードはハッシュ化されている（bcrypt等）
- [ ] 通信はHTTPS（TLS 1.2以上）
- [ ] 不要な個人情報をログに記録していない
- [ ] APIレスポンスに不要な内部情報を含めていない

## シークレット管理
- [ ] APIキー・パスワードはGCP Secret Manager / GitHub Secretsで管理
- [ ] `.env`ファイルが`.gitignore`に含まれている
- [ ] `docs/secrets-management.md`の手順に従っている

## 依存パッケージ
- [ ] 既知の脆弱性のあるパッケージを使用していない
- [ ] `dependabot.yml`が設定されている

## GCP設定
- [ ] サービスアカウントに最小権限が付与されている
- [ ] 不要なAPIが無効化されている
- [ ] Cloud Armorの設定（必要に応じて）
